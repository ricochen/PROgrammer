You've chosen to learn back-end, start by watching this video to learn how to start a server

Now download and install Node and npm and follow this guide to install Express

After you have Node, npm, and Express installed, create your own local server and have it up and running on port 3000

In your server, make a get request to google.com to retrieve the HTML, and show it on your localhost

Anytime you need help, refer to the Express documentation
